# Views - Practice Problems

## Problem 1: Simple Views
Create views for:
1. Active customers (status = 'active')
2. High-value products (price > $500)
3. Recent orders (last 30 days)
4. Employee directory (name, email, department)

## Problem 2: Views with JOINs
Create views combining:
1. Orders with customer information
2. Products with category details
3. Employees with their department and manager
4. Sales with product and customer info

## Problem 3: Aggregate Views
Create views showing:
1. Sales summary by month
2. Top 10 customers by revenue
3. Product inventory levels
4. Department headcount and average salary

## Problem 4: Materialized Views
Create and refresh materialized views for:
1. Daily sales report (expensive aggregation)
2. Customer lifetime value
3. Product popularity rankings
4. Regional sales statistics

## Problem 5: Updatable Views
Create updatable views with:
1. WITH CHECK OPTION to ensure data meets view criteria
2. Update data through the view
3. Insert new records through view
4. Delete records through view

## Problem 6: Security Views
Create views that:
1. Hide sensitive columns (salary, SSN)
2. Filter by user's department
3. Mask email addresses
4. Show only own records
